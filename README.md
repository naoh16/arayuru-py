# arayuru-py
Python3/PyQtGraph(PyQt4)/PyAudioで作った読み上げ音声収録ツール

## Requirement

    $ conda install pyqt=4 pyqtgraph
    $ python -m pip install pyaudio

## Usage

    $ python arayuru.py sample.txt
